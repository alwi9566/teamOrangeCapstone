# RAVEN - Marketplace Finder Chrome Extension

A Chrome extension that automatically appears on Craigslist, eBay, and Facebook Marketplace to help you track listings.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select this project folder
5. Visit Craigslist, eBay, or Facebook Marketplace to see RAVEN appear in the bottom-left corner

## Features

- Automatically injects on marketplace sites
- Filter listings by platform (All, Craigslist, Ebay)
- View listing statistics (count and average price)
- Browse top listings with images and prices
- Close button to hide the overlay

## Usage

The RAVEN interface will automatically appear in the bottom-left corner when you visit:
- Any Craigslist site (*.craigslist.org)
- eBay (*.ebay.com)
- Facebook Marketplace (facebook.com/marketplace)
